= API 문서

== 예외 처리 가이드

기본적으로 HTTP 상태 코드를 준수하여 예외를 처리합니다.

4xx 예외 발생 시, body 값으로 디버깅을 위한 메시지를 제공합니다.

```
{
    "message": "계획된 카테고리 정보가 없습니다."
}
```

== 강제 버전 업그레이드용 헤더

*Request Header*
```
// iOS 의 경우
—header 'App-Version: iOS/0.5'

// Android 의 경우
—header 'App-Version: Android/0.5'
```

만약 클라이언트가 최신 버전이 아닌 경우, 426 Upgrade Required 응답을 반환합니다.

link:https://developer.mozilla.org/ko/docs/Web/HTTP/Status/426[426 Upgrade Required (RFC 2817)]
